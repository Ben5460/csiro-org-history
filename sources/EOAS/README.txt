Explanation for this XML data from Gavan John Mccarthy <gavan.mccarthy@unimelb.edu.au>, Melbourne Uni. 

The entity data you are seeking, with all the other EOAS entities, is not just encoded in html but also in XML using the EAC schema.
See: http://eac.staatsbibliothek-berlin.de/

This data is available through our OAI-PMH repository:
http://oai.esrc.unimelb.edu.au/EOAS/provider

The following command (verb) will give you all the records:
http://oai.esrc.unimelb.edu.au/EOAS/provider?verb=ListRecords&metadataPrefix=eac
